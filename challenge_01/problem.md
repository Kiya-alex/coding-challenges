# Challenge 01: Count Word Frequency

## Problem
Write a Python function that takes a string of text and returns
a dictionary with each word as a key and its frequency as the value.

## Example

Input:
"the cat sat on the mat the cat"

Expected Output:
{"the": 3, "cat": 2, "sat": 1, "on": 1, "mat": 1}

## Constraints
- Ignore capitalization ("The" and "the" count as the same word)
- Ignore punctuation
- Input will always be a non-empty string
