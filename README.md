# Coding Challenges

Self-contained coding challenges with problem statements,
starter templates, and fully tested solutions.

## Challenges

| # | Title | Difficulty |
|---|-------|------------|
| 01 | Count Word Frequency | Easy |

## Structure

Each challenge folder contains:
- problem.md — problem statement and examples
- solution_template.py — starter code
- solution.py — complete solution with tests

## Run a solution

python challenge_01/solution.py

## Technologies
- Python 3
- Unix command-line environment
